## Workflow notes for next audit

1. The Bayer congress portal is Akamai-gated; programmatic curl returns a bot-block HTML page. Headless Chromium with a single HCP-attestation click clears the gate for the session.
2. The Regeneron Medical Congresses portal exposes posters only behind an HCP attestation that hides direct PDF URLs (JS-bound download buttons). Manual HCP attestation per file is required.
3. The Genentech `medically.gene.com` portal mixes `/unrestricted/` (HTML landing) and `/restricted/` (gated PDF) paths; some `/restricted/` PDFs returned 200 without auth in this run.
4. PMC `/articles/PMC.../pdf/` URLs were challenged by a JavaScript proof-of-work; the workaround is to resolve the underlying DOI and pull from the publisher (link.springer.com `/content/pdf/...`, journalretinavitreous.biomedcentral.com `/counter/pdf/...`, JAMA via the `watermark02.silverchair.com` redirect).
5. EyePoint rebranded from eyepointpharma.com to eyepoint.bio; the legacy scientific-publications page is 404 and PDFs live in `wp-content/uploads/YYYY/MM/`.
6. Ocular Therapeutix publications archive lives at the investors subdomain (`investors.ocutx.com/scientific-medical-presentations`), not the corporate site.
7. Outlook Therapeutics, Intas/Sun, Biocon, Lupin host no publication library on their corporate sites; rely on PubMed / PMC for new wet-AMD content.
